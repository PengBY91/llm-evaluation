# 大模型评测任务管理平台 - 前端

基于 Vue3 + Element Plus 的前端应用。

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

应用将在 http://localhost:3003 启动。

## 构建

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

## 功能

- **评测任务管理**: 创建、查看、删除评测任务，实时查看任务进度，下载评测结果
- **任务配置管理**: 创建和管理评测配置模板，快速复用常用配置
- **数据集管理**: 浏览、添加、删除数据集，查看数据集详情和样本
